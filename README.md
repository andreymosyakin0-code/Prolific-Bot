# Prolific-Bot
A simple auto accept script for studies in prolific (website version). It automatically accepts studies in prolific and can reload the cite when the study is full/ it cannot accept it. Has a prolific sound alert which you can test and has a console tab where it shows the time it accepted the study and what it did recently. Has a userfriendly menu with all different options you might need for prolific which you can also move around and hide it with an arrow at top right corner. Feel free to switch to another tab, just dont close the one with this bot in prolific or it wouldnt work! When it finds a study it will automatically accept it and switch it to the tab with prolific.
If you find any bugs/glitches or if you know how I can impove my script you can write it in feedback!
To dowload/use the script you need to click "install this script" or go to code tab and copy (ctrl/cmd A) the text that is written in "source code" and paste it into tampermonkey (I like to use it and it is tested there) and it should work.
If the menu doesnt pop up in https://www.prolific.com/ or in https://app.prolific.com/studies you should reload the cite or watch a tutorial how to use tampermonkey or if it still doesnt work you should write it in feedback and I'll help!
I'm pretty sure its undetectable!
